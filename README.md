# TaskFlow

**TaskFlow** is a fully responsive **task management web application** that helps teams organize and track tasks efficiently. The project is built using **React.js** and **Tailwind CSS**, with **Local Storage** used for storing and managing task data in the browser.

## 📌 About the Project

TaskFlow is designed to simplify task management between **admins and employees**. The admin can create tasks, assign them to employees, and track their progress. Employees can view their assigned tasks and update their status accordingly.

The application provides a clear overview of **active, completed, and failed tasks**, helping both admins and employees manage work efficiently.

## 🚀 Features

### 🔐 Authentication

* **Login and Logout system**
* Separate workflow for **Admin and Employee**

### 👨‍💼 Admin Features

* Create new tasks
* Assign tasks to employees
* View all assigned tasks
* Monitor task status:

  * Active Tasks
  * Completed Tasks
  * Failed Tasks

### 👨‍💻 Employee Features

* View tasks assigned by the admin
* See detailed task information including:

  * Task Title
  * Date
  * Category
  * Description
* Accept assigned tasks
* Mark tasks as **Completed**
* Mark tasks as **Failed**

### 📊 Task Tracking

Employees can clearly see:

* Total **Active Tasks**
* Total **Completed Tasks**
* Total **Failed Tasks**

### 📱 Responsive Design

* Fully responsive interface
* Works smoothly on **mobile, tablet, and desktop**

### 💾 Data Storage

* **Local Storage** is used to store task data and user information in the browser.

## 🛠️ Tech Stack

* **React.js** – Frontend framework
* **Tailwind CSS** – Styling and responsive UI
* **JavaScript (ES6+)**
* **Local Storage API** – Client-side data storage

## 📂 How the System Works

1. Admin logs in to the system.
2. Admin creates a task with **title, date, category, and description**.
3. The task is assigned to a specific employee.
4. Employees can view all assigned tasks in their dashboard.
5. Employees can **accept, complete, or mark a task as failed**.
6. The task status updates accordingly in the system.

## 🎯 Purpose of the Project

The goal of **TaskFlow** is to build a **simple and effective task management system** that helps teams track task progress and manage responsibilities in an organized way.
